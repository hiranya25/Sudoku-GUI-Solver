# Sudoku-GUI-Solver


This project is a Sudoku solver implemented using the **backtracking algorithm**.  
It includes both a **graphical user interface (GUI)** and a **text-based version** of the solver.

---

## How to Run

Run the graphical version with:

```bash
python GUI.py
